# GitIM v0.8.9

## 新功能
- Agent 支持 per-agent Claude effort 配置（`--effort`），可按 agent 调节推理强度
- Provider 新增 Claude Opus 4.8 作为一等模型选项
- Daemon status 现在暴露 push backlog 数量与 auth 熔断状态，排查同步问题更直观

## 修复
- sync_loop auth 熔断改为 half-open 恢复：不再永久锁死，PAT 更新后可自动恢复 git 同步
- 杀掉 daemon 后清理残留的 pid / socket 文件，避免 stale 文件干扰下次启动
