# GitIM v0.7.1

## 新功能

- 归档 DM 支持懒加载、前缀搜索和分页（Load more）
- 卡片讨论有新动态时前端弹 toast 提醒
- sync 层新增 `recover_from_stale_rebase`，支持默认分支解析
- sync 层引入类型化 `GitError::DetachedHead`
- `/git/init` 自动忽略 agent memory 和本地 artifact
- pi provider 暴露 `auto_retry_end` 失败信息

## 修复

- runtime 在重启竞态后能正确恢复 daemon
- runtime recovery 不再静默继承全局 `user.name`
- codex provider 从 rollout token count 推导上下文窗口
- sync 循环顶部自愈卡住的 rebase / detached HEAD 状态
- `abort_rebase` 返回 Ok 前会校验 rebase 标记已清除
- pi provider 按真实 pi-mono RPC event schema 解析
- 前端侧边栏在内容溢出时保持在视口内

## 改进

- runtime 测试通过 `GITIM_LOG_DIR` 环境变量隔离 daemon 日志，HomeGuard 统一到 common 模块
- pi provider 的 tool-result 提取对齐项目约定
