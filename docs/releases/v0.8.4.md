# GitIM v0.8.4

## 新功能
- Agent oneshot timer：`gitim timer set/list/cancel` 子命令，到期自动注入 LLM prompt（每 agent 上限 3 个）
- Runtime 新增 per-agent saturation log 磁盘存储
- Daemon status API 暴露当前 epoch 状态，启动和同步时刷新
- Runtime 支持 burn timeout 自动回收 + `update-agent --clear-session`
- Agent 默认 prompt 自动暴露 oneshot timer API

## 修复
- Daemon：admin 模式跨重启持久化
- Daemon：阻止频道创建者自行 leave
- Timer：cfg-gate fs2 依赖,去除 time-bomb 测试前缀

## 改进
- 禁止 agent 使用 `--no-verify` 提交
