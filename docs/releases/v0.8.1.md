# GitIM v0.8.1

## 新功能

- 新增 Team Flows v1：通过 `flows/<slug>/index.md` 模板定义 DAG 工作流，支持 `gitim flow list/show/create/rm/validate` 子命令
- Team Flows v1.5：新增 run 状态机与 channel 绑定，CLI 支持 `flow start/runs/run-show/node-set/run-cancel`
- WebUI 新增 Flows tab：mermaid DAG 可视化 + react-markdown 渲染 + Recent runs 列表
- WebUI 频道顶部新增 ChannelActiveRuns 活动 run 横条，新增 RunDetail 详情页
- Agent 默认 prompt 暴露 Flows API 与 run lifecycle 契约
- 卡片视图展示 agent 自我介绍
- 频道支持展开全部 @mentions
- 聊天界面新增临时消息送达回执

## 修复

- 消息回执过滤掉当前用户自身
- 修复长看板状态渲染
- 修复移动端看板列表溢出
- 修复 fleet agent 活动渲染
- daemon flow not-found 错误返回 404；非法 run status filter 返回结构化 error_code
- watcher 仅对 `index.md` 编辑发出 FlowModified
- 零节点 run 自动完成；flow 删除时联动清理 `runs/`
- core 校验 flow node ID 字符集
- `NodeStatus` / `RunStatus` 事件字符串统一走 `as_str()`，保证 wire 一致

## 改进

- 延长 add-agent preflight 超时
- 默认前端密度调优
- agent control plane 布局更紧凑
- 优化 agent fleet 刷新
- workspace usage 分组展示优化
- 引入 workspace 级 clippy lints + pre-commit hook（cargo fmt + clippy 强制）
