# GitIM v0.8.3

## 新功能
- Provider usage 上报拆分,各 provider 可声明独立的累计/增量语义
- gitim-core 新增 epoch metadata 类型与 YAML 解析器
- 频道显示创建者与路由 chips
- Agent 工作状态与在线状态拆分展示

## 修复
- hermes: 正确解析 ACP usage 字段别名
- opencode: 累计 step usage 计费,避免漏算
- opencode: `step_finish(reason=stop)` 立即完成 turn,不再等待进程退出
- pi: 将 tool-use turn 计入 usage 统计
- runtime: provider 失败后自动重试 agent turn
- daemon: 保留序列化错误信息,避免吞错
- daemon: 修复 self-join 合规问题
- frontend: 侧边栏正确突显未读频道
- frontend: 自动选中聊天时保留未读状态
- frontend: 优化聊天滚动与未读状态行为

## 改进
- 全 workspace clippy 清理:消除 gitim-daemon / gitim-runtime / gitim-sync / gitim-index 中的 unwrap/expect
- opencode 测试串行化并放宽超时,提升并行稳定性
