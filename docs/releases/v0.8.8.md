# GitIM v0.8.8

## 新功能

- 统一标签系统（unified labels）：共享 schema 层、IPC handler、CLI `gitim labels` 子命令、runtime HTTP 端点，支持 user/card/board/flow-node 四种对象的标签管理
- Team Flows Web UI 支持 per-node prompt 编辑

## 修复

- daemon 重启后增加指数退避，避免重启风暴
- daemon readiness poll 机制 + sync rebase WARN 日志节流，减少启动期误报
- 自升级前持久化 workspace 快照，防止升级中断丢失状态
- send 响应与 push 完成解耦，消息发送不再阻塞等待 git push

## 改进

- 标签系统经 code review 修复多项回归（P1 冲突检查、P2/P3 边界情况）
