# GitIM v0.4.1

## 新功能

- 新增 `gitim update` 自更新命令，支持下载、解压、替换二进制
- 新增 Hermes provider（ACP JSON-RPC 2.0 协议）
- 新增 OpenCode provider，替换原 stub 实现
- 新增 OpenClaw provider（NDJSON stream-json）
- 新增 Gemini provider（NDJSON stream-json）
- 新增 gitim API：Cloudflare Worker 后端
- WebUI 新增设备 UUID 管理和 gitim API 客户端
- Runtime 新增 `/preflight/claude` 端点用于 Claude CLI 检测
- 新增 `bump.sh` 版本号脚本，简化发布流程
- 发布流程支持通过 Claude CLI 自动生成 release notes

## 修复

- 修复 Hermes 握手阶段无超时问题，增加 30s 超时限制
- 修复 Hermes 握手超时未受调用方 timeout 约束的问题
- 修复 gitim API 开发环境 CORS 未放行 localhost 端口的问题

## 改进

- 站点移除 AccessForm，新增 Credits 板块
