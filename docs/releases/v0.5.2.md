# GitIM v0.5.2

## 新功能
- Agent 配置可编辑：AgentDetail 页支持编辑 `system_prompt` / `env` / `.env` 文件，带未保存守卫和保存反馈
- `PATCH /workspaces/{slug}/agents/{id}` 端点支持 system_prompt（merge 语义）、env（整体替换 + key 校验）、`.env` 文件写入（0600 权限、64KB 上限）
- Workspace 初始化时自动把 `.env` 加入 `.gitignore`（幂等，`system@gitim` 作者）
- AgentCard / AgentDetail 展示 provider · model 信息,新增 ProviderBadge 组件

## 修复
- agent 活动浮层逃离 sidebar overflow 裁剪
- 校准 DAU 统计口径,统计数据定期刷新
- agent detail 正确展示 Provider,移除 Sonnet 默认值误导
- dev 构建的 binary 放到 `~/.gitim/bin`,符合自动更新路径要求
- `.gitignore` helper 遇到 `.env` 负向规则时安全退出

## 改进
- 抽取 `EnvVarsEditor` 共用组件
- 使用人数文案调整为"今天已使用过"
- vite chunk size 警告阈值提升到 800kB
