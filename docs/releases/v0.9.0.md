# GitIM v0.9.0

## 新功能

- **Channel Project 分组**：channel 可选归入 project，侧栏按 project 折叠分组、cards 支持按 project 过滤；纯管理语义，不影响 routing / 权限 / 归档 / flows
- **Epoch 自动轮转**：分支 commit 数超阈值（默认 1M）自动开新 epoch 分支并封存旧分支，原子 push 双 ref 仲裁单 winner，保证消息零丢失
- **Display name 渲染**：WebUI 全面显示每个用户的 display_name（消息、线程、输入框、@mention、DM 标题、侧栏、成员列表）
- **Flow 节点结构编辑**：WebUI 内直接增删改 flow 节点（runtime `PUT /im/flows/{slug}`）
- **Fleet SSH 隧道自愈**：per-node watcher 检测隧道中断并强制重启

## 改进

- Agent detail 页对运行中被锁定的字段显示 edit-mode banner
- daemon-web 对已封存 epoch 分支只读拦截，永不发布到 sealed branch

## 修复

- 浏览器模式下 channel 成员变更时刷新成员列表
- 默认走 gitim CORS proxy
- Epoch 轮转 migrate replay 失败时自动恢复，fence 对 corrupt epoch.yaml fail-closed
- `ChannelSummary` 响应正确暴露 project 字段
