# GitIM v0.9.1

## 新功能
- Kimi Code provider 支持 CLI 0.14 ACP 协议，并从 `wire.jsonl` 读取用量信息
- WASM 导出 `validateHandler`，并修复 wasm32 下的随机数后端支持
- Flow 校验会对 Phase 2 占位节点类型给出警告

## 修复
- 修复回复路由未继承祖先消息 mentions 的问题
- 修复 runtime 更新快照锁处理，避免并发状态不一致
- 修复 Kimi provider 关闭流程、用量上下文和 home override 相关问题
- 修复 fleet SSE 订阅重连，改为指数退避避免频繁重连

## 改进
- daemon-web 的 `.thread` 解析、格式化、冲突处理和元数据逻辑统一走 WASM 实现
- 统一 runtime/daemon 相关配置常量与测试 GitHub API base env 命名
- 统一 agent 文件大小上限为 64KB 配置常量
